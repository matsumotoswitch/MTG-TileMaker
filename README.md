# MTG-TileMaker

投稿サイト用にMagic: The Gathering (MTG) の画像をタイル状に合成するためのWebツールです。

**[開く](https://matsumotoswitch.github.io/MTG-TileMaker/)**

## 概要 (Overview)

MTG-TileMakerは、ブラウザ上でMTGの画像を検索し、Webサイト投稿用の画像を作成するツールです。

## 使い方 (Usage)

1.  [MTG-TileMaker](https://matsumotoswitch.github.io/MTG-TileMaker/) にアクセスします。
2.  画面の指示に従ってMTGのカード画像の選択や設定を行います。
3.  生成された結果を保存または印刷します。
