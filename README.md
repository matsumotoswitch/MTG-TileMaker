# MTG-TileMaker
## 概要 / Overview
MTG-TileMakerは、Scryfall APIを利用してMagic: The Gatheringのカード画像を検索し、noteなどのブログ投稿に最適なタイル状のコラージュ画像をブラウザ上で作成できるWebユーティリティです。
単なる画像の並べ替えだけでなく、ブログの横幅に合わせたサイズ調整や、直感的な操作でのレイアウト作成をサポートします。

## 主な機能 / Features
*  **Scryfall API連携検索:** 日本語・英語での検索、部分一致・完全一致に対応。
*  **言語切り替え:** 同セット・同番の別言語版カードへワンクリックで切り替え可能。
*  **直感的なレイアウト:** ドラッグ＆ドロップによるカードの追加、並び替え、削除。
*  **柔軟な画像操作:** カードの90度回転、ローカル画像のアップロード（白背景の透過処理機能付き）。
*  **詳細な出力設定:** 横の配置枚数、カード幅、カード間の余白（ギャップ）、出力画像の全体幅、配置（左・中央・右）の調整。

## 使い方 / Usage
1. [MTG-TileMaker](https://matsumotoswitch.github.io/MTG-TileMaker/) にアクセスします。
2. 左側のパネルからカード名を検索し、結果から右側のキャンバス（ドロップエリア）へカードをドラッグ＆ドロップします。
3. キャンバス上でカードの並び替えや回転を行い、設定パネルで出力サイズや枚数を調整します。
4. 「画像生成」ボタンをクリックすると、レイアウトされた画像がPNG形式でダウンロードされます。
